reverse proxy server
Проксі сервер може бути прямим і зворотим. Прямий проксі перенаправляє всі запити клієнта (інтернет браузерів) до потрібного сайту так, наче він є і клієнтом.
Така його властивість використовується для забезпечення конфіденційності, безпеки. Також його можна використати для доступу до заборонених ресурсів під іншим ІР,
економії трафіку, контролю трафіку і т.д.
Зворотній проксі (Reverse proxy) навпаки використовується не клієнтом, а сервером. Фактично клієнт працює з проксі сервером (хоча може і не знати цього), який
перенаправляє запити на інші вебсервери. Він працює з конкретними вебсерверами, для яких налаштований. Зворотний проксі сервер балансує навантаження між його вебсерверами, 
захищає від атак, приховує від клієнтів реальні характеристики вебсерверів, зменшує навантаження на основні сервери за рахунок кешування контенту, забезпечує більш 
швидкий доступу до ресурсу шляхом зжимання інформації для передачі на клієнти.
